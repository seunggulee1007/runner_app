🏃‍♀️ 러닝 트래커 앱 PRD

1.  제품 개요 제품명(가칭): StrideNote 목적: 사용자가 달리기를 할 때
    거리, 속도, 심박수, 러닝 패턴을 자동 기록하고, 개인의 성장과
    피드백을 직관적으로 보여주는 앱. 단순한 기록이 아닌 “러닝 스토리”를
    만들어주는 개인 맞춤형 트래커.

2.  주요 사용자 페르소나

-   초보 러너: 건강을 위해 러닝을 시작했지만 꾸준히 이어가기 어려운
    사람.
-   취미 러너: 주 2~3회 러닝하며 기록 향상에 관심이 있는 사용자.
-   경쟁 러너: 대회 준비나 기록 향상에 집중하는 사용자.

3.  주요 기능 3.1 코어 기능

-   러닝 자동 기록 (GPS 기반 거리, 페이스, 시간, 고도 추적)
-   심박수 연동 (웨어러블 기기 연동)
-   훈련 요약 리포트 (달리기 후 자동 생성)
-   러닝 히스토리 (주/월간 통계 시각화 + 배지 시스템)

3.2 부가 기능 - 러닝 플랜 추천 - 소셜 공유 기능 - 음악 연동

4.  사용자 여정

5.  앱 실행 → “오늘의 러닝 시작하기”

6.  “러닝 시작” 클릭 → GPS 연결 + 카운트다운

7.  달리기 중 → 실시간 데이터 표시 + 음성 알림

8.  종료 → 자동 저장 + 리포트 생성

9.  분석 보기 → 통계 대시보드 이동

10. 목표 설정 → AI 플랜 생성

11. 기술 요구사항

-   플랫폼: iOS, Android (Flutter/React Native)
-   서버: AWS Lambda + DynamoDB
-   API: HealthKit, Google Fit, Spotify, Kakao Share
-   보안: OAuth2 인증, HTTPS 암호화

6.  성공 지표 (KPI)

-   DAU 10,000명
-   세션 평균 25분 이상
-   목표 달성률 60% 이상
-   리텐션(30일) 40% 이상
-   앱 평점 4.5점 이상

7.  로드맵 Phase 1 (MVP, 0~3개월): 기본 기록 + 리포트 Phase 2 (3~6개월):
    AI 플랜 + 배지 시스템 Phase 3 (6~12개월): 커뮤니티 + 챌린지 + 음악
    연동

8.  디자인 방향성

-   블루 톤 기반의 역동적 컬러
-   한 손 조작 중심의 직관적 UI
-   즉각적 피드백 UX
